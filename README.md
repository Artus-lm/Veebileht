# Võrkpalliklubi Structura/Tartu vald veebileht

See on (pool)ametlik veebileht, koondamaks kogu info Eesti esiliigas mängiva võrkpalliklubi Structura/Tartu vald kohta. Veebileht on vajalik, et kogu info klubi kohta oleks fännidele ühest kohast leitav. Veebileht on loodud aine "Sissejuhatus erialasse" veebiprojekti raames.

## Ülesehitus

Meie veebileht koosneb neljast alalehest: 
- Avaleht 
    - olnud ja tulevased mängud
- Klubi ja mängijate tutvustus
- Registreerimisvorm
- Test – milline positsioon sobib sulle?

## Disain

Veebilehe disainis kasutame klubi mänguvormil olevaid värve.

##  Kasutatud tehnoloogiad
- HTML
- CSS
- JavaScript
- JSON-failid
- webscraping
